# lein-beandock

A Leiningen plugin to use Docker containers with AWS ElasticBeanstalk. Built on top of lein-beanstalk and lein-uberimage.

## Usage

Put `[lein-beandock "0.1.0-SNAPSHOT"]` into the `:plugins` vector of your project.clj.
